vagrant/ubuntu-bionic
=====================

An Ubuntu Bionic Beaver development environment featuring:

- LightDM
- AwesomeWM
- command-line utilities for:

  - network management and scanning
  - system administration
  - archive management
  - ...

- dotfiles
